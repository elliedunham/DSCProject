About the Model

The win probability model is a logistic regression model. My model has an accuracy score of 87%. All variables are considered significant.

win probability = MinutesRemaining + homeNumOfWinsBefore:homeNumOfGamesBefore+ visitorNumOfWinsBefore:visitorNumOfGamesBefore + homeTeamScoreDif + homeShotMinCount:awaySavesMinCount + homeSavesMinCount:awayShotMinCount + awayCornerMinCount + homeSavesCount:awayShotCount + homeShotCount:awaySavesCount + homeCardCount + awayCardCount + homeCornerCount + awayCornerCount + saaGame

Factors considered:

  • Minutes left in the game
  • Number of games won and games played by the home team before the game
  • Number of games won and games played by the away team before the game
  • Home team score differential at each minute
  • Number of home team shots each minute (total and per minute)
  • Number of away team shots each minute (total and per minute)
  • Number of home team saves each minute (total and per minute)
  • Number of away team saves each minute (total and per minute)
  • Number of home team corners each minute (total and per minute)
  • Number of away team corners each minute (total and per minute)
